## 功能描述
1. 添加命令 "调整所有文本节点高度" - 用户按需执行
2. 扫描当前 canvas 所有文本节点
3. 根据内容计算合适高度（使用设置中的 textNodeMaxHeight 作为最大值）
4. 超出最大高度时，节点内容可滚动

## 需要修改的文件

### 1. src/canvas/canvas-manager.ts
添加方法 `adjustAllTextNodeHeights()`：
- 获取当前 canvas 文件
- 遍历所有文本节点
- 使用 `calculateTextNodeHeight()` 计算高度
- 更新节点高度并保存文件
- 返回调整的节点数量

### 2. src/main.ts
注册新命令：
```typescript
this.addCommand({
    id: 'adjust-all-text-node-heights',
    name: 'Adjust All Text Node Heights',
    callback: () => this.canvasManager.adjustAllTextNodeHeights(),
});
```

### 3. styles.css
添加滚动样式（仅对超出最大高度的节点）：
```css
.canvas-node .canvas-node-content {
    overflow-y: auto;
}
```

## 实现细节
- 宽度保持设置中的 `textNodeWidth`（默认400px）
- 高度根据内容计算，但不超过 `textNodeMaxHeight`（默认800px）
- 只调整纯文本节点（排除图片、附件等）
- 执行后显示通知，告知调整了多少个节点

## 性能考虑
- 只在用户主动执行时运行
- 时间复杂度 O(n)，n 为节点数量
- 数千个节点在现代设备上几秒内完成
